# proto-plugins

plugins for [proto](https://moonrepo.dev/proto)

## plugins

### claude-code

[claude-code](https://github.com/anthropics/claude-code)

```bash
$ proto plugin add claude-code https://raw.githubusercontent.com/sasaplus1/proto-plugins/main/claude-code/plugin.toml
$ proto install claude-code
```
